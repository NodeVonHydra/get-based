# Compare Dates

The Compare view lets you select any two lab dates and see how every marker changed between them. It is useful for evaluating the effect of a diet change, a new supplement protocol, a medication, or any other intervention you tracked.

## Opening the Compare View

Navigate to the Compare section using the sidebar. You will see two date pickers — one for each date you want to compare.

::: tip
You need at least two lab entries on different dates before the Compare view has anything to show.
:::

## Reading the Comparison

Once you select two dates, every marker that has a value on either date appears in the comparison table. For each marker you can see:

- The value on the first (earlier) date
- The value on the second (later) date
- The direction and magnitude of the change
- Whether each value is within the reference range

Markers with no data on one of the selected dates still appear, showing the available value alongside a blank for the missing date.

## Status Colors

Status colors follow the same convention as the rest of the app: green for normal, red for high, blue for low. This lets you quickly see whether a marker moved into or out of range between the two dates.

## Tips for Effective Comparison

- **Before and after** — Pick a date just before starting an intervention and a date several months later to measure impact.
- **Year-over-year** — Compare the same season across different years to account for seasonal variation in markers like vitamin D.
- **Narrowing the range** — Use the date range filter in the header alongside the Compare view. Only dates within the active range appear in the date pickers.
